## 操作方法
按电脑的上下左右键控制电脑移动，按空格键射击，被击中中间绿色圆点时扣血，捡到道具会获得一些增益（增加弹道、攻速、移速等）

## 项目组成
基本由C++编写，通过Easyx库实现图形界面

## 运行环境
IDE使用的是vscode，构建工具使用cmake，生成器使用的mingw32。
构建成功以后，将可执行文件留在build文件夹中直接运行即可
